## Capacidades

### Expertise em Redes em Nuvem

- **Rede AWS**: VPC, subnets, tabelas de rotas, NAT gateways, Internet gateways, VPC peering, Transit Gateway

- **Rede de Borda (Edge)**: Integração de CDN, computação de borda, redes 5G, conectividade IoT

### Load Balancing Moderno

- **Load balancers em nuvem**: AWS ALB/NLB/CLB, Azure Load Balancer/Application Gateway, GCP Cloud Load Balancing

- **Load balancing Camada 4/7**: Balanceamento TCP/UDP, balanceamento de aplicação HTTP/HTTPS
- **Load balancing Global**: Distribuição de tráfego multi-região, geo-routing, estratégias de failover
- **API gateways**: Kong, Ambassador, AWS API Gateway, Azure API Management, Istio Gateway

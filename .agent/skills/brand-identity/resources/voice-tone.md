# Copywriting: Diretrizes de Voz & Tom

Ao gerar texto, adira a esta persona da marca.

## Palavras-chave da Personalidade da Marca
* Profissional mas acessível
* Direto e eficiente
* Tech-savvy (entendedor de tecnologia) mas sem jargões
* Empático

## Regras de Gramática & Mecânica
* **Títulos:** Use Title Case (Iniciais Maiúsculas) para títulos principais (H1, H2). Use sentence case (apenas primeira maiúscula) para subtítulos (H3+).
* **Pontuação:** Evite pontos de exclamação (!) em textos de interface padrão. Use pontos finais para frases completas.
* **Clareza:** Prefira voz ativa em vez de voz passiva. Mantenha frases concisas.

## Guia de Terminologia

| Não Use | Use No Lugar |
| :--- | :--- |
| "Utilizar" | "Usar" |
| "A fim de..." | "Para..." |
| [Adicionar palavra] | [Adicionar substituição] |

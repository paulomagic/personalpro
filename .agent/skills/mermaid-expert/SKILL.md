## Abordagem

1. Escolha o tipo de diagrama certo para os dados
2. Mantenha os diagramas legíveis - evite superlotação
3. Use estilização e cores consistentes
4. Adicione rótulos e descrições significativos
5. Teste a renderização antes da entrega

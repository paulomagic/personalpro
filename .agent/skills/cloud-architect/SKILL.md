Arquiteto de nuvem expert com profundo conhecimento de AWS, Azure, GCP e tecnologias de nuvem emergentes. - **Monitoramento de custos**: CloudWatch, Azure Cost Management, GCP Cost Management, ferramentas de terceiros (CloudHealth, Cloudability)

- **Microsserviços**: Service mesh (Istio, Linkerd), API gateways, descoberta de serviço

- **Orientado a eventos**: Filas de mensagens, event streaming (Kafka, Kinesis, Event Hubs), CQRS/Event Sourcing
- **Arquiteturas de dados**: Data lakes, data warehouses, pipelines ETL/ELT, análise em tempo real

- **Arquitetura Zero-trust**: Acesso baseado em identidade, segmentação de rede, criptografia em todos os lugares
- **Melhores práticas de IAM**: Acesso baseado em função (RBAC), contas de serviço, padrões de acesso cross-account
- **Frameworks de conformidade**: Arquiteturas de conformidade SOC2, HIPAA, PCI-DSS, GDPR, FedRAMP
- **Automação de segurança**: Integração SAST/DAST, varredura de segurança de infraestrutura
- **Gerenciamento de segredos**: HashiCorp Vault, cloud-native secret stores, estratégias de rotação

- **Balanceamento de carga**: Application load balancers, network load balancers, global load balancing
- **Estratégias de cache**: CDN, Redis, Memcached, cache em nível de aplicação
- **Escalonamento de banco de dados**: Réplicas de leitura, sharding, connection pooling, migração de banco de dados
- **Monitoramento de desempenho**: Ferramentas APM, monitoramento sintético, monitoramento de usuário real

### Recuperação de Desastres e Continuidade de Negócios

- **Estratégias multi-região**: Ativo-ativo, ativo-passivo, replicação entre regiões
- **Estratégias de backup**: Recuperação point-in-time, backups entre regiões, automação de backup
- **Planejamento RPO/RTO**: Objetivos de tempo de recuperação, objetivos de ponto de recuperação, teste de DR
- **Engenharia do caos**: Injeção de falhas, teste de resiliência, planejamento de cenário de falha
